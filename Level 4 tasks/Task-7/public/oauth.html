What is OAuth?

OAuth is an authorization framework that allows users to grant applications limited access to their resources without sharing passwords.

Access Token

An Access Token is a temporary credential that applications use to access protected resources on behalf of a user.

Refresh Token

Refresh Tokens are used to obtain new Access Tokens without requiring the user to log in again.

Authorization Server

The Authorization Server verifies user identity and issues tokens to authorized applications.

OAuth Authorization Flow

1. User Requests Login
→
2. Authorization Server
→
3. Access Token Issued
→
4. API Access Granted
